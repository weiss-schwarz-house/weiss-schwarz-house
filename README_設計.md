# サイト設計メモ（ヴァイスシュヴァルツの館）

## 目的
- ボードゲームのルールブック・要点を素早く参照できるようにする
- サークル内での共有・参照を主目的とする
- 将来的に多数のゲーム追加を想定する

## 想定コンテンツ
- 全体トップ
- ゲーム個別ページ（1ゲーム1ページ）
- ルールPDF（外部リンク）
- ゲームごとの要点・気づき・Q&A

## ページ構成
- index.md（全体トップ）
- games/index.md（ゲーム一覧）
- games/xxx.md（個別ゲーム）

※ 個別ゲームはすべて games 配下に配置する

## ゲーム個別ページの考え方
- 1ゲームにつき1Markdownファイルとする
- 各ゲームページの中に以下を含める
  - ルールブックPDFへのリンク
  - ルール要点（要約）
  - 勘違いやすい点・気づき

## 分類軸（暫定）
- プレイ人数
- プレイ時間
- 重さ（軽 / 中 / 重）
- ジャンル（協力 / 正体隠匿 など、複数指定可）

※ 分類はフォルダ分けではなく、ゲーム情報として管理する

## ナビゲーション
- ゲーム一覧ページを起点に各ゲームへ遷移する
- 将来的にサイドバーでゲーム一覧を表示する
- 将来、検索機能を追加する可能性あり

※ 初期段階では検索機能は実装しない

## ルールPDFの扱い
- ルールPDFは外部ストレージに配置する
- ゲーム個別ページからリンクして参照する

## 運用方針
- 新規ゲーム追加時は、個別ゲームページを1つ追加するのみ
- ゲーム数が増えても、一覧や分類の手動修正が最小限になる設計を目指す
